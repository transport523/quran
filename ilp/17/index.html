17.The Night Journey

  1. Soti so (Allah a) Piyakalalakao Niyan so Oripun Niyan (a Muhammad) ko Kagagawi-i pho-on ko Masjidil Al-Haram si-i ko Masjidil Al-Aqsa, so piyakandakulan Nami sa mapiya so muliliyota-on,- ka Phakaila-in Nami Sukaniyan ko manga Tanda Ami: Mata-an! A so Allah so Puphakanug, a Puphakailai
